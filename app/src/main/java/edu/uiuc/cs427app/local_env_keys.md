In the local.properties file, add the following keys:
```
GEMINI_API_KEY=
MAPS_API_KEY=
WEATHER_INFO_API_KEY=
```

GEMINI_API_KEY is a Free Tier Google AI Studio API key. https://aistudio.google.com/app/api-key

MAPS_API_KEY is a Google Maps API key.

https://developers.google.com/maps/documentation/android-sdk/get-api-key
https://developers.google.com/maps/documentation/address-validation/get-api-key

WEATHER_INFO_API_KEY is a Weather API key. https://www.weatherapi.com/signup.aspx
